## All installation methods

### Using Homebrew

You can use [Homebrew](https://brew.sh/) on macOS or Linux:

```bash
brew install fzf
```

### Using Mise

You can use [mise](https://github.com/jdx/mise) to install fzf:

```bash
mise use -g fzf@latest
```

### Using git

Clone the repository and run the install script:

```bash
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install
```

The install script will:
- Add fzf to your `$PATH`
- Set up shell integration (key bindings and fuzzy completion)
- Prompt you for configuration options

<Tip>
  This method automatically sets up shell integration for you.
</Tip>

### Binary releases

Download the latest binary for your platform from the releases page:

- [https://github.com/junegunn/fzf/releases](https://github.com/junegunn/fzf/releases)

Extract the binary and place it in your `$PATH`:

```bash
# Example for Linux
tar -xzf fzf-*-linux_amd64.tar.gz
sudo mv fzf /usr/local/bin/
```

## Setting up shell integration

Shell integration provides key bindings (CTRL-T, CTRL-R, ALT-C) and fuzzy completion for your shell.

<Note>
  If you installed fzf using git, shell integration is already configured. Skip this section.
</Note>

Add the following line to your shell configuration file:

<Tabs>
  <Tab title="Bash">
    Add to `~/.bashrc`:
    
    ```bash
    # Set up fzf key bindings and fuzzy completion
    eval "$(fzf --bash)"
    ```
  </Tab>
  
  <Tab title="Zsh">
    Add to `~/.zshrc`:
    
    ```zsh
    # Set up fzf key bindings and fuzzy completion
    source <(fzf --zsh)
    ```
  </Tab>
  
  <Tab title="Fish">
    Add to `~/.config/fish/config.fish`:
    
    ```fish
    # Set up fzf key bindings
    fzf --fish | source
    ```
  </Tab>
</Tabs>

<Note>
  The `--bash`, `--zsh`, and `--fish` options are available in fzf 0.48.0 or later.
</Note>

### Older versions or manual setup

For finer control or older versions, source individual script files from the `/shell` directory:

```bash
# Find the location of the files
apt show fzf  # On Debian/Ubuntu
brew info fzf # On macOS with Homebrew
```

### Disabling specific key bindings

You can disable CTRL-T, CTRL-R, or ALT-C bindings by setting the corresponding variable to empty:

<CodeGroup>

```bash Bash
# Disable CTRL-R and ALT-C
FZF_CTRL_R_COMMAND= FZF_ALT_C_COMMAND= eval "$(fzf --bash)"
```

```zsh Zsh
# Disable CTRL-R and ALT-C
FZF_CTRL_R_COMMAND= FZF_ALT_C_COMMAND= source <(fzf --zsh)
```

```fish Fish
# Disable CTRL-R and ALT-C
fzf --fish | FZF_CTRL_R_COMMAND= FZF_ALT_C_COMMAND= source
```

</CodeGroup>

<Warning>
  Set the variables before sourcing the script. Setting them after will have no effect.
</Warning>

## Vim/Neovim plugin

If you use [vim-plug](https://github.com/junegunn/vim-plug), add this to your Vim configuration:

```vim
Plug 'junegunn/fzf', { 'do': { -> fzf#install() } }
Plug 'junegunn/fzf.vim'
```

- `junegunn/fzf` provides the basic library functions
- `fzf#install()` ensures you have the latest binary
- `junegunn/fzf.vim` provides useful commands

<Tip>
  For Neovim users who prefer Lua-based plugins, check out [fzf-lua](https://github.com/ibhagwan/fzf-lua).
</Tip>

## Upgrading fzf

Upgrade fzf using the same method you used to install it:

```bash
# Git installation
cd ~/.fzf && git pull && ./install

# Homebrew
brew update && brew upgrade fzf

# MacPorts
sudo port upgrade fzf

# Chocolatey
choco upgrade fzf

# Vim-plug
:PlugUpdate fzf
```

## Verifying installation

After installation, verify that fzf is working:

```bash
# Check version
fzf --version

# Test basic functionality
ls | fzf

# Test shell integration (if configured)
# Press CTRL-R in your terminal
```

<Tip>
  Run `fzf --help` to see all available options, or `fzf --man` to view the full manual.
</Tip>
